\subsection{\-Detailed \-Description}
\-Parse a stream of \-J\-S\-O\-N data. 

\-Using this class directly you can reduce the apparent latency for each download/parse cycle of documents over a slow connection. \-You can start parsing $\ast$and return chunks of the parsed document$\ast$ before the entire document is downloaded.

\-Using this class is also useful to parse huge documents on disk bit by bit so you don't have to keep them all in memory.
