00034 // Returns a string containing a nicely formatted stack trace.
00035 // 
00036 // This function gets the stack trace for the current thread. It will
00037 // be from the caller of GTMStackTrace upwards to the top the calling stack.
00038 // Typically this function will be used along with some logging, 
00039 // as in the following:
00040 //
00041 //   MyAppLogger(@"Should never get here:\n%@", GTMStackTrace());
00042 //
00043 // Here is a sample stack trace returned from this function:
00044 //
00045 // #0  0x00002d92 D ()  [/Users/me/./StackLog]
00046 // #1  0x00002e45 C ()  [/Users/me/./StackLog]
00047 // #2  0x00002e53 B ()  [/Users/me/./StackLog]
00048 // #3  0x00002e61 A ()  [/Users/me/./StackLog]
00049 // #4  0x00002e6f main ()  [/Users/me/./StackLog]
00050 // #5  0x00002692 tart ()  [/Users/me/./StackLog]
00051 // #6  0x000025b9 tart ()  [/Users/me/./StackLog]
00052 //
